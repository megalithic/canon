## Canon

Taking your various social media bookmarks and favorites and aggregating them into one searchable, filterable location.


#### Supports

- [ ] Instagram (saved)
- [ ] Twitter (liked)
- [ ] Facebook (saved)
- [ ] Slack (starred)
- [ ] Reddit (saved)
- [ ] GitHub (starred)


#### Features

- [ ] User-configurable API credentials
- [ ] Searchable
- [ ] Filterable
- [ ] Offline-friendly (PWA)
- [ ] Taggable
- [ ] Add notes/metadata
- [ ] Mobile-friendly


#### Technologies

* Elixir
* Phoenix
* Postgres
* GraphQL
* Elm
